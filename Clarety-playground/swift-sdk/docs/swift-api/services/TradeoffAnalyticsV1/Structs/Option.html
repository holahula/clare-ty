Option

public struct Option: JSONEncodable, JSONDecodable

An option in a decision problem (i.e. a row in a tabular representation of the data).

  • key

    An identifier for the option that is unique among all options for the problem.

    Declaration

    Swift

    public let key: String
  • values

    Option-specific values for the columns (objectives) defined for the problem. Specify a dictionary of column keys to option values. Value requirements vary by column type; a value must be of the type defined for its column. An option that fails to specify a value for a column for which isObjective is true is marked as incomplete and is excluded from the resolution. Example: ["Name": .Text("BRZ"), "Price": .NumericInt(27395)]
